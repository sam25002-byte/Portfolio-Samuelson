Reflection:
a few of the most challenging parts of this project, besides not being familiar with github and general github procedure would be working with other people who also doent know. What I mean by that is having issues using the same css and knowing coding best practices so that you can work well with teams. because we arent familiar with this, there was a lot of struggling where one person made a trait too broad, and since I wasnt comfortable changing their stuff, had to work around it. If I were to do this again, I would want to work with people I am more familiar with so that I can better give suggestions and communicate with them instead of a complete stranger who just happened to take the same class as me. Overall though, it has been pretty fun and it is good to know more about what real-world coding is meant to look like by doing it wrong.
